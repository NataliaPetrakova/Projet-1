La Balise Img

Description

La balise ‹img› sert à ajouter une image, c'est un élément vide il n'y a donc pas de balise de fermeture. Il faut l'insérer dans une balise paragraphe ‹p› et l'associer à différents attributs. (Voir tableau ci-dessous.)

Tableau des attributs

Attribut(s) Désignation Unité(s) acceptée(s)
src Précise au navigateur l'emplacement du fichier. url
alt Description textuelle de l'image (utile aux personnes aveugles). texte
title Infobulle au survol de l'image. texte
width Largeur d'origine de l'image pixel
height hauteur d'origine de l'image pixel

Exemple

‹p› ‹img src="images/chatoncoquin.jpg" alt="Photo de chaton coquin sur un clavier" title=" Chaton coquin clavier" width ="509px" height="337px" /› ‹p›
