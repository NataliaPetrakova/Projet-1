Liste non-ordonnée ul

Description

La balise "Ul" permet de faire une liste non-ordonnée (par défaut une liste à puces), chaque élément de la liste est contenu dans une balise "li".

On peut aussi réaliser des listes imbriquées (pour des tables de matières, faire des listes de liens hiérarchisées).

Les autres éléments HTML relatifs aux listes : ‹ol›, ‹li› , ‹menu›.

A savoir : on peut modifier l'apparence des puces avec le CSS.

Exemple

‹ul›
‹li› premier élément ‹/li›
‹li› deuxième élément ‹/li›
‹li› troisième élément ‹/li›
‹/ul›

Résultat

  • élément 1
  • élément 2
  • élément 3
