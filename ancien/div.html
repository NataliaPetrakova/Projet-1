Div

Cette balise définit une division ou une section, appelé aussi calque, dans un document HTML. Elle est utilisée pour regrouper des éléments sous forme de bloc et ainsi pouvoir les formater en CSS. (Par exemple on peut contenir tous les éléments de l'en tête).

Attribut(s) Désignation Unité(s) acceptée(s)
id Identification unique d'un élément. texte
class Identification de plusieurs éléments. texte

Exemple

‹div id ="header"› ‹div id ="nav"› ‹div id ="main"› ‹div id ="footer"›
