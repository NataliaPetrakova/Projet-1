La balise HEADER

L’élément ‹header› définit l’en-tête d’un document ou d’une section. Il peut être utilisé pour contenir une introduction ou des liens de navigation. Vous pouvez avoir plusieurs éléments ‹header› dans un même document. Attention : Un élément ‹header› ne peut pas être inclus à l’interieur des éléments suivants : ‹footer›, ‹address› et ‹header›.

Attribut(s) Désignation Unité(s) acceptée(s)
id nomme un élément texte
class attribut une classe nom de la classe

Exemple

‹article›
‹header›
‹h1›Bonne année‹/h1›
‹p›‹time pubdate datetime="2017-02-28"›‹/time›‹/p›
‹/header›
‹p›Février 2017.‹/p›
‹/article›

Résultat

Bonne année

Février 2017.
