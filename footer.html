La balise FOOTER

Description

La balise ‹footer› définit un pied de page d’un document ou d’un élément ‹article›.
Un élément ‹footer› doit contenir des informations sur l’élément auquel il appartient. Un pied de page contient généralement l’auteur du document, les informations de copyright, des liens vers des conditions d’utilisation, informations de contact, etc …
Vous pouvez avoir plusieurs éléments ‹footer› dans un seul document.

Tableau des attributs

Attribut(s) Désignation Unité(s) acceptée(s)
id nomme un élément texte
class attribut un classe nom de la classe

Exemple

‹footer›
‹p›Auteur : OFP‹/p›
‹p›Tous droits réservés‹/p›
‹p›Publié le 28 Février 17‹/p›
‹/footer›

Résultat

Auteur : OFP / Tous droits réservés / Publié le 28 Février 17
